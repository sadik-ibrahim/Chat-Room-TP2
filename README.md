# TP2 – Chat em Tempo Real com Flet

Trabalho Prático 2 de Computação Móvel
Aplicação de chat em tempo real desenvolvida em Python com a framework Flet.

## Como correr

```bash
# Criar e ativar ambiente virtual
python -m venv .venv
.venv\Scripts\activate  # Windows
# source .venv/bin/activate  # Linux/Mac

# Instalar dependências
pip install -r requirements.txt

# Correr a aplicação
python main.py
```

## Correr como app web

```bash
python main.py --web
```
